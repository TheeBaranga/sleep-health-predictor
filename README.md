# sleep-health-predictor
About This Project
This repository contains a full machine learning lifecycle project aimed at predicting sleep disorders (such as Insomnia and Sleep Apnea) based on daily habits, cardiovascular health, and demographic data.

Utilizing the Kaggle Sleep Health and Lifestyle Dataset, this project walks through data cleaning, feature engineering, and model training to achieve high-accuracy predictions. The final machine learning model is served through a custom Flask web interface, allowing users to input lifestyle metrics and receive real-time sleep health predictions.

Tech Stack: Python, Pandas, Scikit-Learn, Flask, HTML/CSS.
